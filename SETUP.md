# Инструкция первого запуска
## Делай строго по порядку — каждый шаг занимает 2-5 минут

---

## Шаг 1 — Установить Python (5 минут)

Python — это двигатель бота. Без него код не работает.

1. Открой браузер → перейди на **https://www.python.org/downloads/**
2. Нажми большую жёлтую кнопку **"Download Python 3.12.x"**
3. Запусти скачанный файл
4. ⚠️ **ВАЖНО:** поставь галочку **"Add Python to PATH"** (внизу окна установщика!)
5. Нажми "Install Now"
6. Дождись завершения (1-2 минуты)

**Проверка:** открой новое окно терминала в VS Code (Terminal → New Terminal) и напиши:
```
python --version
```
Должно появиться: `Python 3.12.x`

---

## Шаг 2 — Установить зависимости бота (2 минуты)

В терминале VS Code выполни (убедись что ты в папке my-first-project-1):
```
cd C:\Users\Muslima\my-first-project-1
pip install -r requirements.txt
```
Будет идти скачивание. Подожди до строки "Successfully installed..."

---

## Шаг 3 — Создать бота в Telegram (5 минут)

1. Открой Telegram → найди **@BotFather**
2. Напиши: `/newbot`
3. Имя бота (как отображается): `Бизнес-стабилизатор`
4. Username (техническое, на латинице, заканчивается на bot): например `biznes_stab_bot`
5. BotFather пришлёт **токен** вида: `7123456789:AAFxxxxxxxxxxxxxxxxxxxxxxxx`
6. **Скопируй этот токен** — он нужен на следующем шаге

---

## Шаг 4 — Узнать свой Telegram ID (2 минуты)

1. В Telegram найди **@userinfobot**
2. Напиши ему `/start`
3. Он ответит: `Your user ID: 123456789`
4. **Скопируй это число**

---

## Шаг 5 — Создать файл настроек (2 минуты)

В терминале VS Code:
```
copy .env.example .env
```

Затем открой файл `.env` в VS Code и замени значения:
```
BOT_TOKEN=7123456789:AAFxxxxxxxxxxxxxxxxxxxxxxxx   ← вставь свой токен
ADMIN_ID=123456789                                  ← вставь свой ID
DAILY_HOUR=9
DAILY_MINUTE=0
PAYMENT_PROVIDER_TOKEN=
SUBSCRIPTION_PRICE=29900
```

Сохрани файл (Ctrl+S).

---

## Шаг 6 — Проверить что всё готово (1 минута)

```
python setup_check.py
```

Если все пункты с ✅ — переходи к следующему шагу.

---

## Шаг 7 — Первый запуск! 🚀

```
python bot.py
```

В терминале появится:
```
2026-07-01 09:00:00 | INFO | Бот запущен.
```

Открой Telegram → найди своего бота → напиши `/start`

Ты должен увидеть первый вопрос онбординга! 🎉

---

## Если что-то пошло не так

| Ошибка | Решение |
|--------|---------|
| `python` не найден | Переустанови Python с галочкой "Add to PATH", перезапусти VS Code |
| `ModuleNotFoundError` | Запусти `pip install -r requirements.txt` |
| `Unauthorized` | Токен бота неправильный, проверь .env |
| Бот не отвечает | Проверь что python bot.py запущен и работает |
