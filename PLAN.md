## ЭТАП 1 — Запуск (цель: бот отвечает в Telegram)
**Время: 15-20 минут**
**Результат: бот живой, можно написать /start**

### Шаг 1.1 — Проверить Python
```bash
python --version
```
Нужна версия 3.10 или выше.

### Шаг 1.2 — Установить зависимости
```bash
cd my-first-project-1
pip install -r requirements.txt
```

### Шаг 1.3 — Создать .env файл
Скопировать .env.example → .env
Заполнить BOT_TOKEN и ADMIN_ID.

### Шаг 1.4 — Получить токен бота (делает человек, 5 мин)
1. Открыть Telegram → найти @BotFather
2. Написать /newbot
3. Придумать имя: «Бизнес-стабилизатор»
4. Придумать username: @biznes_stabilizator_bot (или любой свободный)
5. Скопировать токен вида 123456789:AAF...

### Шаг 1.5 — Узнать свой Telegram ID (делает человек, 1 мин)
1. Найти @userinfobot в Telegram
2. Написать /start
3. Скопировать числовой ID

### Шаг 1.6 — Первый запуск
```bash
python bot.py
```
✅ Проверка: написать боту /start в Telegram → должен прийти первый вопрос онбординга
