# Бизнес-стабилизатор — Telegram подписной бот

Telegram-бот, который ежедневно даёт 1–2 конкретных действия по работе с клиентами и продажами,
помогая структурировать бизнес-процессы и повысить системность взаимодействия с клиентской базой.

---

## Структура проекта

```
my-first-project-1/
├── bot.py           # Бот: онбординг, команды, подписка, платежи
├── config.py        # Настройки из .env
├── database.py      # SQLite: пользователи, подписки, онбординг
├── scheduler.py     # Ежедневная рассылка (APScheduler)
├── messages.py      # 15 фокусов по работе с клиентами
├── requirements.txt # Зависимости Python
├── .env.example     # Шаблон переменных окружения
└── README.md        # Этот файл
```

---

## Быстрый старт

### 1. Создайте бота в Telegram
1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot`
2. Придумайте имя и username
3. Скопируйте токен

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Настройте .env
```bash
cp .env.example .env
```
Откройте `.env` и вставьте токен:
```
BOT_TOKEN=ваш_токен
ADMIN_ID=ваш_telegram_id
```

### 4. Запустите бота
```bash
python bot.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Онбординг (5 вопросов) + первый фокус |
| `/today` | Получить сегодняшнее действие |
| `/subscribe` | Оформить подписку |
| `/progress` | Прогресс и статус подписки |
| `/stop` | Отписаться от рассылки |
| `/help` | Список команд |
| `/stats` | Статистика (только для админа) |
| `/cancel` | Прервать онбординг |

## Онбординг (5 шагов)

При первом запуске пользователь проходит диагностику:
1. Описание сферы деятельности (свободный текст)
2. Количество активных клиентов (кнопки)
3. Способ учёта клиентов (кнопки)
4. Частота инициации контакта (кнопки)
5. Главная сложность в работе с клиентами (кнопки)

По результатам — определяется категория ситуации и предлагается первый фокус.
После онбординга — предложение подписки.

## Модель доступа

| Статус | Доступ |
|--------|--------|
| Новый пользователь | Онбординг + 3 бесплатных дня |
| Без подписки (после 3 дней) | Ограниченный онбординг, paywall |
| Активная подписка | Полный доступ ко всем ежедневным действиям |

Цена: **299 ₽/месяц** через Telegram Payments.

## Подключение платёжного провайдера

1. Напишите [@BotFather](https://t.me/BotFather) → `/mybots` → выберите бота → **Payments**
2. Подключите провайдера (Stripe, ЮКасса, Robokassa и др.)
3. Скопируйте токен провайдера в `.env`:
   ```
   PAYMENT_PROVIDER_TOKEN=ваш_токен_провайдера
   ```

Для тестирования используйте тестовый токен провайдера — реальные деньги не списываются.

## Ежедневная рассылка

- Время: **9:00 МСК** ежедневно (настраивается в `.env`)
- Получают: все подписанные пользователи, прошедшие онбординг
- Пользователям без подписки после 3 дней — напоминание о подписке

---

## Юридически безопасные формулировки

Бот использует только допустимые формулировки:
- ✅ «структурирование работы с клиентами»
- ✅ «оптимизация клиентских касаний»
- ✅ «систематизация продаж»
- ✅ «поддержка ежедневных бизнес-действий»

Запрещённые формулировки не используются:
- ❌ «вернёшь доход», «заработаешь»
- ❌ «гарантируем результат»
- ❌ «найдём деньги»

## Добавление уроков

Откройте [messages.py](messages.py) и добавьте в список `LESSONS`:

```python
{
    "focus": "📋 Фокус #16 — Название",
    "body": "Описание проблемы без обещаний результата...",
    "action": "⚡ Действие:\n1. ...\n2. ...\n3. ...",
    "timeframe": "30 минут",
},
```
